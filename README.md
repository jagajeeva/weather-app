# Weather App

A simple Weather Forecast Application built using Python and Tkinter

## Features

* Search weather by city
* Temperature in Celsius
* Feels Like temperature
* Humidity
* Pressure
* Wind Speed
* Weather Condition

## Technologies Used

* Python
* Tkinter
* OpenWeatherMap API

## How to Run

1. Install Python
2. Replace `YOUR_API_KEY` with your OpenWeatherMap API key
3. Run:

```bash
python weatherapp.py
```
